Software Designer

📍 India ⏲ Part time 👤 Designer

Job Description:

Duties and Responsibilities

  • Create, develop, and install new software systems.
  • Designing apps.
  • Analyzing and changing software to improve performance and remove errors.
  • Use different programming languages to create or fix software.
  • Upgrade interfaces as required.
  • Plan and assist with software installations or repairs.
  • Communicate with clients or users on software usability and maintenance.
  • Work with other designers, developers, and managers.

Minimum Qualifications

  • Bachelor's Degree in Computer Science, Information Systems, Engineering or related field or equivalent work experience.
  • Excellent communication skills to effectively communicate and explain technical matters or instructions to the client & team.
  • Expertise in Python, C, C++, C#, HTML, CSS, Ruby, PHP, SQL.
